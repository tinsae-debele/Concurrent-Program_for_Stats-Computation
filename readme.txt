--------------------------------------
Semaphore-based Number Orderer
Authors: Tinsae Debele 
--------------------------------------

Instructions:
1) Open terminal, and navigate to the path where STATS.c is located.
2) In terminal, type "gcc STATS.c -o run" and press enter to compile.
3) Type "./run" and press enter to run the program.
4) Enter y to run the debugger mode, or n to skip.
5) When prompted, enter 5 INTEGER numbers separated by a space. For the first test case, enter "5 6 8 2 7". The ordered result will be returned and displayed.
6) Enter "./run" again to re-execute the program.
7) Enter the second test case, "10 9 11 5 7". The re-ordered numbers will be returned.
